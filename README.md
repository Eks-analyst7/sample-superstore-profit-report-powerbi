# Sample Superstore Profit Report using Power BI

## Overview

This project is an interactive Power BI dashboard developed using the Sample Superstore dataset.

The dashboard provides insights into sales, profit, customer segments, regions, product categories, and sub-categories to help understand overall business performance.

## Dashboard Preview

<img width="100%" alt="Dashboard Screenshot" src="Screenshots/profit_report_dashboard.png">

## Objectives

- Analyze overall business performance.
- Identify profitable regions and customer segments.
- Track profit trends over time.
- Discover top-performing product categories.
- Build an interactive business intelligence dashboard.

## Tools Used

- Power BI

- Data Modeling
- Data Visualization

Sample Superstore Dataset

- Order ID

- Sub-Category

- Profit

## Dashboard Components

### KPI Cards

- Total Sales: 2.30M
- Total Profit: 286.40K
- Total Quantity: 38K

### Visualizations

- Profit by Year and Quarter
- Profit by Region
- Profit by Segment
- Profit by Category
- Profit by Sub-Category
- Matrix Analysis

### Filters

- State slicer

## Key Insights

- West region generated the highest profit.
- Consumer segment contributed significantly to profits.
- Technology category performed the best.
- Copiers and Phones were among the top-performing products.
- Profit trends varied across years.
